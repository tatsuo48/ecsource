# ecsresource

ECS タスク定義の CPU、メモリ、メモリ予約といったリソースを見やすく表示するツールです。

```bash
$ ecsresource ファイル名
+------------------------------------------------------------------------+-----+--------+------------+
|                                  NAME                                  | CPU | メモリ   | メモリ予約   |
+------------------------------------------------------------------------+-----+--------+------------+
| タスク全体(タスクレベルにメモリ予約の設定値はないのでメモリの値を流用)                 | 512 |   1024 |       1024 |
| datadog-agent                                                          |  50 |      0 |         50 |
| nginx                                                                  |   0 |      0 |        200 |
| webapp                                                                 |   0 |      0 |        424 |
| nr_infra_agent                                                         |   0 |      0 |        100 |
| nr_apm_daemon                                                          |   0 |      0 |        100 |
| log_router                                                             |   0 |      0 |        100 |
| 合計                                                                    |  50 |      0 |        974 |
| 余り                                                                     | 462 |   1024 |         50 |
+------------------------------------------------------------------------+-----+--------+------------+
```

## インストール

```bash
make install
```
